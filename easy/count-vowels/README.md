# Count Vowels

**Difficulty:** Easy

## Problem
Count the number of vowels in a given string.

### Example
Input: `"hello world"`  
Output: `3`

### Constraints
- String length ≤ 10⁴  
- Only English letters and spaces

### Test Cases
| Input | Expected Output |
|:------|:----------------|
| "hello world" | 3 |
| "AEIOU" | 5 |
| "xyz" | 0 |
| "" | 0 |
| "ChatGPT rocks!" | 3 |

### Complexity
- **Time:** O(n)  
- **Space:** O(1)
